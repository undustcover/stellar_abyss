# 独立生成 Public 测试：TEST-XXXX

## 一、测试目标

本测试不再从既有作者修改稿反推答案，而是验证现有作者审美决策库和笑话写作方法论库能否独立产生作者可继续加工的 Public 版本。

标准闭环：

```text
作者提供 Draft
→ 系统依据当前正式库独立生成 System Public V1
→ 冻结 System Public V1
→ 作者基于 V1 修改为 Author Public V2
→ 比较 V1／V2
→ 作者校准修改原因
→ 更新正式库
→ 下一轮独立测试
```

## 三、输入隔离声明

### 系统生成前允许读取

- 本轮作者提供的 Draft；
- 当前有效的作者审美决策库；
- 当前有效的笑话写作方法论库；
- Draft 明确引用且理解事实所必需的章节上下文；
- 当前章节必须保持的事实、人物、POV 和信息边界。

### 系统生成前禁止读取

- 本章任何作者目标稿、作者预改稿或隐藏参照稿；
- 本章未来的 Author Public V2；
- 历史案例中的作者原句、成品笑点、具体桥段和段落顺序；
- 旧对比报告、旧修改意见和能够泄露本章目标答案的材料。

### 隔离检查

- [ ] 本章不存在已被系统读取的作者目标稿
- [ ] 历史案例只通过正式抽象库进入本轮
- [ ] 如必须扩展上下文，已记录扩展文件及原因
- [ ] 未以“作者可能会怎么改”为目标复刻旧章表达
