# 作者审美决策库

## 一、使用边界

本库保存作者确认或待验证的审美判断，不保存作者原句、范文段落、成品桥段、笑点、比喻、动作链、段落顺序或固定结尾方案。

本库用于当前场景审美诊断和审美编辑裁决，不直接向正文模型提供历史案例。正文模型只接收当前诊断后形成的问题、边界、选定方向和创作自由区。

## 二、判断状态

|级别|含义|能否作为硬约束|
|---|---|---|
|L1 单案例假设|来自一个案例，原因仍可能有歧义|否，只能提示风险|
|L2 重复倾向|在多轮或多章重复出现|通常不能，需结合适用条件|
|L3 作者确认原则|作者明确确认并通过跨场景验证|仅在适用条件成立时可以|

## 六、同质化监测

每完成若干章后检查：

- [ ] 不同章节是否反复采用同一种场面结构
- [ ] 不同人物是否反复使用同一种幽默机制
- [ ] 结尾是否总以同一种含混或压力方式收束
- [ ] “少解释”是否退化成信息缺失
- [ ] “保留张力”是否退化成故意不写清楚
- [ ] “人物体面”是否阻止人物真正失控或犯错
- [ ] L3 规则是否在不适用场景仍被当成硬约束

发现同质化时，优先收窄规则适用条件，不直接新增相反规则。
