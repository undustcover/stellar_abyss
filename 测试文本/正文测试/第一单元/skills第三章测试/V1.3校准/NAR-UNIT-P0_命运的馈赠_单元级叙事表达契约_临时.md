# P0《命运的馈赠》单元级叙事表达契约（临时）

当前用途：

章节级契约应只记录本章相对本文档的差异、收紧、放开或修正。

## 三、单元核心表达

P0《命运的馈赠》是一个由误会喜剧逐步转向现实反噬的前置单元。

它的表层是连续误会喜剧：人物荒唐、对话错位、情境离谱，读者在不知道完整真相时也能获得喜剧感。

它的暗面是现实证据链：前半段所有看似热心、帮忙、到场、问流程、管闲事的行为，后半段都会被现实语言重新组织成对马木不利的材料。

它的底层是轻微异常：异常世界不能正面解释，只能通过话术、节奏、留座、资格感、集体一致性、服务过度贴合等方式轻轻露头，并且总体被喜剧压住。

P0 的最终效果不是让读者觉得“马木终于得到了机会”，而是让读者理解：

> “命运的馈赠”在 P0 中不是礼物，而是陷阱。

## 四、单元结构节奏

|阶段|章节范围|主要读感|表达功能|
|---|---|---|---|
|误会启动|第1-2章|会所误会喜剧|建立马雯雯对马木的不靠谱滤镜和云水汇误会|
|异常回声|第2章后 / 第3章前|短促、冷、不解释|给读者一点类型承诺，但不让马木和马雯雯掌握异常|
|误会升级|第3-10章|喜剧持续升级，异常被压住|让读者在错误框架里感到事情越来越怪，逐步积累现实证据痕迹|
|喜剧转冷|第11章|前期笑点被现实语言重组|派出所来电，前10章的热闹变成材料|
|真相翻面|第12-13章|误解变证据|陈科线责任外移，马木从帮忙者变成疑似参与流程者|
|全线崩盘|第14-15章|笑点反杀，情绪沉底|小满线、陈科线、马木旧伤共同推动 P1 心理动机成立|

## 五、读者信息层级

### 单元通用原则

读者信息不应一次性打开。

P0 的读者体验依赖三层错位：

1. 读者先和马雯雯一起误会，以为这是会所喜剧。
2. 读者逐渐感觉事情不止会所，但具体真相的显影程度必须由章节级契约逐章确认。
3. 第11章以后，读者才看见前期笑点如何变成现实材料。

### 信息显影总表

|信息类型|前半单元默认处理|后半单元处理|备注|
|---|---|---|---|
|云水汇真实性质|默认不直接说明|可逐步被现实流程和证据链解释|具体显影点由章节级契约决定|
|诈骗/资金/项目结构|默认不得自动显影|第11章后可由正式调查和现实后果显影|第3-10章不能因大纲旧表述自动写明，必须经章节级契约确认|
|异常机制|不得解释|仍不完整解释|只允许通过异常回声、话术、留座、资格感等表层露头|
|马木旧事|前期只露边角|第15章打开更深伏笔|不能提前洗白|
|证据链|前期作为喜剧动作存在|第11章后翻译成材料|前期不得用旁白说明“这是证据”|

## 六、POV 通用原则

### 默认 POV

P0 大多数章节默认以马雯雯的认知过滤为主。

她的功能不是发现真相，而是把读者带入误会系统。读者能领先她多少，必须由章节级契约决定。

### 特例

第2章后 / 第3章前的“异常回声”可以绕开马雯雯视角，用短促冷视角给读者类型承诺。

该特例不得自动扩展到其他章节。其他章节若需要非马雯雯 POV，必须在章节级契约中单独确认。

### 马雯雯 POV 边界

|可承担|不可承担|
|---|---|
|家庭滤镜、职场压力、会所误会、对马木不信任、对荒唐场面的吐槽|提前识破诈骗结构、解释异常机制、替读者做全知判断、突然越过本人经验完成破案|

## 七、单元误读系统

误读系统是 P0 的主要叙事发动机。

章节级契约应继承本节通用误读框架，只记录本章特有误读映射。

|后台真实|前台误读|单元功能|
|---|---|---|
|马木介入陈科事件|马木和云水汇有不正经联系|建立不靠谱堂哥滤镜|
|云水汇包装陈科|高级会所服务、贵宾接待、特殊体验|制造会所误会和高级感错位|
|陈科被尊重感托住|中老年人被服务哄舒服、嘴硬要面子|制造孔乙己式体面喜剧|
|马木处理现实风险|懂灰色场所规矩、爱管闲事、管得很熟|制造“民间办事处”喜剧，并埋证据链|
|陈艾艾保护父亲|女儿来抓人、接管现场、硬顶父亲|让正常现实追问托出陈科荒唐|
|陈灏感知异常或识别假高端|会所熟客、富二代混子、嘴欠探店者|用喜剧压住异常判断|
|异常的留座/资格感影响|服务太好、话术太统一、位置太被重视|让读者觉得怪，但不解释机制|
|前期帮忙痕迹|热心、荒唐、到处插手|第11章后翻译为现实材料|

## 八、表达成分配比基准

强度：无 / 低 / 中 / 高 / 核心。

|表达成分|单元基准|说明|
|---|---|---|
|喜剧|核心|前10章必须好笑，不能只依赖作者知道真相|
|现实压力|中，后半升高|前期埋在生活与流程里，第11章后显影|
|异常感|低到中|必须可感知，但总体被喜剧压住，不写成设定说明|
|信息揭示|前低后高|前期以误读和错接为主，后期由现实调查和反噬推进|
|留白|中|旧事、异常机制、马木真实创伤不能写满|
|情绪压迫|前低后高|第11章后逐步转冷，第14-15章沉底|
|关系变化|中|马雯雯对马木的判断逐渐松动，但不能过早洗白|

章节级契约可以调整某章配比，但必须说明相对本基准的差异。

## 九、人物单元功能基准

本节只记录人物在 P0 单元中的叙事功能，不替代人物卡。

|人物|单元功能|通用边界|
|---|---|---|
|马雯雯|主要视角过滤者，误会系统入口|不能提前识破诈骗或异常；不能突然越过 POV|
|马木|被误读的不靠谱堂哥，热心介入者，后续证据链承压者|不能过早被洗白，不能写成专业反诈导师|
|陈科|体面嘴硬的入局者，后续责任外移源头|不能写成纯蠢或纯坏；前期不能主动承认被骗|
|陈艾艾|硬、直、独立的女儿，现实处理者|不能柔弱化；不能承担真相解释器功能；具体信息显影受章节契约限制|
|陈灏|以熟客/混子/嘴欠观察者表层进入，底层触到异常或假高端|不能提前变成异常说明者|
|周睿|旧人旁证，提示马木的人生惯性和旧伤边角|不能提前替马木完整洗白|
|张富贵 / 张凡 / 小满|家庭喜剧线与后期情绪反杀线|前期笑点必须能在第14章翻面|
|任战|隐藏加害者和异常机制源头|前期不解释机制，不让马木明确知道异常|

## 十、异常表达规则

P0 异常必须轻、短、压在现实表层下。

### 允许方式

- 话术过于统一；
- 服务过度贴合；
- 称呼、位置、资格、留座等词语反复产生牵引；
- 灯光、香味、电子屏、座位安排让人隐约不适；
- 陈科对“被尊重”“被留位置”的反应明显过强；
- 陈灏或其他人物觉得假高端背后有点不对，但说法仍可被喜剧解释。

### 禁止方式

- 不写灵能波动；
- 不写异能发动；
- 不写场域影响；
- 不解释世界观；
- 不让正文用旁白告诉读者“这是能力”；
- 不让马木提前知道异常；
- 不让《六合集市》APP在 P0 结尾前出现。

## 十一、喜剧与证据链反杀规则

前10章所有喜剧动作都应具备后续被现实重组的潜力。

|前台笑点|后续反杀方向|
|---|---|
|火柴盒、定位、会所误会|云水汇相关物品和到场线索|
|送证件、帮忙找人|到场协助和流程参与|
|问流程、看材料、打电话|联系工作人员、接触资料、参与处理|
|陈科说“他懂”“让他问”|马木被推到懂流程的位置|
|民间办事处|多线资料和痕迹汇聚到马木身上|

前期不得提前告诉读者“这些会变成证据”。后期才让现实语言重新翻译。

## 十二、标题与主题基准

P0 总标题《命运的馈赠》必须保持反讽方向。

标题语义不是“天降机会”，而是：

> 看似馈赠的东西，可能是陷阱。  
> 看似热心的帮忙，可能留下痕迹。  
> 看似荒唐的笑点，可能变成现实材料。

章节标题可以喜剧化，但不能破坏单元整体的反讽方向。

## 十三、硬性禁止项

1. 不得读取旧 P0C0 创作文档作为本轮 V1.3 校准输入。
2. 不得把旧章节正文、旧细纲、旧执行包当作新契约来源。
3. 不得在前期用旁白解释诈骗、异常机制、任战能力或马木旧事。
4. 不得让马雯雯越过 POV 成为真相解释者。
5. 不得让马木过早成为清醒、专业、帅化的反诈导师。
6. 不得把人物事实卡改造成章节写作提示；人物事实调整必须单独进入人物卡。
7. 不得让章节级契约重复本文档大段通用规则；章节级只写差异。
8. 不得把章节级特殊校正自动升级为单元规则，必须先进入待回流事项。

## 十四、P0C3 共创反哺样本

P0C3 共创已经确认一个重要流程原则：

> P0 大纲可以作为上游，但作者在章节级共创中可以收紧或修正大纲中的读者信息显影。

具体反哺：

1. 原大纲中 P0C3 容易被理解为“读者开始进入反诈显影”。
2. 作者明确修正为：P0C3 读者不能知道诈骗，必须基本贴着马雯雯的会所认知走。
3. P0C3 的“怪”不是更脏、更低级或明显不正常，而是服务非常好、高级得超出预期、周到得让人隐约不适。
4. 陈艾艾本章不能以诈骗/钱/项目显影方式说话，只能以硬、直、接管现场的女儿方式追问人、证件、签字和家属介入。
5. 陈灏本章第一印象应是会所熟客，而不是异常感知者或真相解释者。

这说明后续章节级契约必须保留作者共创修正权，不能机械继承 P0 大纲旧表述。
