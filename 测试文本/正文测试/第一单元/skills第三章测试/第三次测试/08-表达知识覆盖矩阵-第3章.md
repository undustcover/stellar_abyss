# 第三章表达知识覆盖矩阵

## 一、矩阵目的

本文件执行 Task 的 T2，判断现有正式知识是否足以支持第三章各功能片段的 L2—L4 表达重构。

本阶段只回答：

1. 现有知识能否识别问题；
2. 是否已有可执行方法；
3. 缺口属于知识、Skill 还是模板；
4. 下一步应提取哪些候选方法。

本阶段不生成 Public 正文，不将候选方法写入正式知识库。

## 三、覆盖等级

|等级|定义|
|---|---|
|充分|既能判断问题，也有步骤、失败边界和自检，可直接进入片段实验|
|部分|能判断问题，或只有局部方法；仍需补操作步骤或组合规则|
|不足|只有高层原则，无法稳定转换为 L2—L4 重构动作|
|不适用|该知识类别不承担当前问题|

## 四、总体能力判断

|知识层|覆盖判断|说明|
|---|---|---|
|作者审美判断|基本充分|能够判断人物工具化、解释过度、旁白越权、留白失效、信息清单化、主角过度胜利等问题|
|喜剧生成方法|局部充分|对生活经验、物件重解、递进包袱、关键词、误读、拆句和自嘲已有可执行步骤|
|非喜剧表达重构方法|不足|缺少关系驱动对白、信息入口重分配、现场证据、行动证词、认知运动等通用步骤|
|混合基调处理|部分|审美库能判断轻喜、压力和不安的边界，但缺少“同一材料承担多种读感”的执行方法|
|L3 段落重组|不足|现有规则能说“不应怎样”，没有稳定的段落重组工作流和深度验收|
|L4 片段表达重构|不足|局部修复 Skill 有基础边界，但缺少 Public 专用重构卡、装配和三类门禁|
|Public 执行 Skill|缺失|当前没有专门承载合格层冻结、片段审计、L0—L5 和修改深度门禁的 Skill|

结论：

> 现有知识支持进入审计和候选方法提取，但不支持直接稳定重跑整章 Public。
