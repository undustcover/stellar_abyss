## 二、L4 片段重构卡

### 2.1 片段功能

- 让马木从手机离手识别风险，但不直接给云水汇定性；
- 让陈科在维护体面时主动泄露更多服务信息；
- 让陈艾艾由追问转为情绪爆发并夺取对话权；
- 让“钱”从父女关系压力中爆发；
- 以陌生来电打断未完成冲突，接入 F05。

### 2.2 必须保留的事实

|事实 ID|冻结事实|
|---|---|
|FACT-F04-01|手机离开陈科约十分钟|
|FACT-F04-02|陈科没有一直看着手机|
|FACT-F04-03|云水汇用“家属不了解、容易打断体验”劝陈科不要告诉家里|
|FACT-F04-04|云水汇询问家庭人数及谁管钱|
|FACT-F04-05|云水汇要求陈科明天继续|
|FACT-F04-06|云水汇认为陈科今天状态不错、适合继续|
|FACT-F04-07|体验贴先贴，之后才签体验确认|
|FACT-F04-08|陈科将服务解释为尊重老人|
|FACT-F04-09|陈艾艾追问实际花费|
|FACT-F04-10|陌生号码来电；陈艾艾先按住手机；陈科认为可能是小刘|

### 2.3 人物目的

|人物|当前目的|对白与动作后果|
|---|---|---|
|陈科|证明自己被专业对待，维护判断力和体面|主动拿服务细节为云水汇辩护，却泄露更多风险信息|
|陈艾艾|确认父亲是否被控制或花钱，同时压住焦虑|从追问变成夺取手机、杯子和话语权，最终直问钱|
|马木|抓住足以改变处理方式的关键矛盾|只问手机离手时间，之后主要让陈科自己说，不完成反诈宣讲|
|马雯雯|在限知范围观察关系与材料变化|不为云水汇定性，只看见同一批“体贴”细节在桌边变了作用|

### 2.4 关系运动

```text
陈科占据解释权
→ 马木只抓手机离手
→ 陈科为自证主动泄露服务细节
→ 陈艾艾发现“家属、钱、明天”同时进入服务范围
→ 父女冲突压过风险盘问
→ 马木退出辩论，不连续取胜
→ 陌生来电让未完成冲突转入下一片段
```

### 2.5 信息入口设计

|入口类型|承担信息|
|---|---|
|关键提问|手机离手约十分钟|
|陈科主动辩护|没有盯着；家属不要打断；询问家庭和钱；明天继续；状态适合继续|
|陈艾艾复述与逼问|确认“别告诉家里”、先贴后签及实际花费|
|物件动作|手机、保温杯和文件袋改变话语权|
|外部事件|陌生来电打断冲突并接入 F05|

### 2.6 禁止改变

- 不新增云水汇真实业务或骗局结论；
- 不让马木成为全知破案者；
- 不改变父女关系和人物选择；
- 不改变马雯雯限知 POV；
- 不改变来电和 F05 的衔接；
- 不新增人物能力、历史或证据。

### 2.7 原表达失败机制

- 马木依次宣读风险问题；
- 陈科逐项合理化；
- 事实以清单顺序进入；
- 马木连续赢下问答；
- 旁白替读者完成风险分类；
- “尊重老人不等于……”形成整齐论证链。

## 三、F04 测试正文

马木从椅背上起来了一点。

“手机给他们多久？”

陈科皱着眉回想：“十来分钟。充个电能出什么事？”

马木把桌边那部手机翻过来。黑屏上那道裂从一角斜到另一角。

“你看着了？”

“我盯着人家干什么？”陈科的声音先高了起来，“小刘就在旁边。人家帮忙，我拿人当贼？”

马木“嗯”了一声，把手机放回去，没再往下问。

他这一停，陈科反而坐不住了。

“再说了，他们什么都提前讲过。体验的时候少跟家里说，省得你们不懂，跑来打岔。问家里几口人、平时谁管钱，也是为了了解生活习惯。调理又不是贴一下就完事，当然叫我明天接着去。”

陈艾艾原本按着保温杯，听到“谁管钱”，手慢慢松开了。

“他们问这个了？”

“问一句怎么了？”陈科把文件袋往自己这边拖，“我今天状态不错，适合继续。人家愿意认真管，你们还不高兴。”

文件袋没拖动。陈艾艾压住了另一角。

“认真管，就是让我别知道？”

“不是别让你知道。”陈科把袋子又拽了一下，“是不让你什么都不懂就瞎掺和。”

“那腰上的贴呢？”陈艾艾没松手，“先贴上，后签字，也因为我不懂？”

“那叫体验确认。”

“体验都完了，确认什么？”

陈科一下没接上。他转向马木，像是终于找到了一个不跟自己吵架的人：“你不是懂这些吗？他们就是流程正规，服务也细。尊重老人有什么错？”

马木看了看被父女俩扯住的文件袋。

“尊重没错。”

陈科的下巴刚抬起来一点。

“手机拿走，家里别说，谁管钱也问。”马木说，“这几件事放一块，我不放心。”

他说到这里就停了，没替云水汇下结论。

陈科脸上的那点胜色挂不住，又不肯落下来：“你们就是把人想坏了。小刘比你们有耐心，问我肩颈，问我睡眠，还提醒我明天过去。你们回来只会审我。”

“我审你？”陈艾艾笑了一下，声音却发紧，“我问到现在，你连花了多少都没说。”

桌边没人接话。

保温杯被她握得咯了一声。

陈科看向她：“我跟你说身体，说服务，说人家专业，你眼里就剩钱？”

“不然呢？”陈艾艾盯着他，“我盯小刘有没有给你盖被子？”

马雯雯低头咳了一声。

没人笑。

陈科的脸僵了僵。他松开文件袋，转手去拿手机，像是这场话已经没必要再说。

手机先在桌上震了起来。

屏幕亮着，没有名字，只有一串号码。

陈科的手刚伸过去，陈艾艾已经按住了手机。

“别接。”

“你干什么？”陈科急了，“万一是小刘？”

“所以才别接。”
