# 第二章第二次修正对比报告

## 一、对比结论

第二版确实执行了一部分第二次问题总结里的方向：它尝试压低马木的连续金句，延后“骗钱的”直接定性，把云水汇写得更像一个正规流程场所，并把“马雯雯误会会所”的前半章方向继续放大。

但第二版也产生了新的问题：它没有稳定做到“筛选、降噪、延后揭示”，而是把部分场景压扁、跳过、旁白变花、人物气质重置。尤其第三幕和带走陈科段落被大幅压缩后，第一版里最关键的父女拉扯、陈科嘴硬、银行卡/席位压力、明天中午保留位置等主钩子被削弱或消失。

因此，本轮不能简单把第二版当作成功样本回写。更准确的判断是：

```text
第二版提供了“该做减法”的方向，但也暴露出“减法不能减掉场面承压点”的流程缺口。
```

## 二、第二版实际修改了什么

|修改面|第二版做法|效果判断|
|---|---|---|
|章节标题|从“他还真去了会所”改为“一通电话”。|降低标题对会所误会的直白提示，但也削弱了章节外部记忆点。|
|开头饭桌|把“热闹得很努力”改成更密集的旁白吐槽和文气句。|解决了原句直白尴尬，但换成了更花的旁白，仍不是场内动作。|
|电话段落|把陈艾艾伪装成更官方、更像公职人员的口吻，增强“家属配合”误会。|方向有效，能让马雯雯误会更体面；但电话信息过度绕行，陈艾艾的真实生活焦急变弱。|
|家庭反应|增加姨父被怀疑、家庭成员互相审视。|能增强生活现场，但有抢戏风险，容易让家庭段笑点过满。|
|云水汇门口|强化正规签到、平板、耳麦、立牌、流程感。|方向正确，比“阴森会所”更有效。|
|陈艾艾出场|把陈艾艾写得更稳、更强势，甚至抽烟、等马木处理。|偏离第二次总结中“慌-羞-火-控”的弧线；她从女儿压力变成了强气场角色。|
|陈科段落|大量删除陈科与陈艾艾、马木的现场拉扯。|减掉了马木金句，但也减掉了陈科的喜剧/刺痛核心。|
|信息披露|不再直接写“骗钱的”，也没有集中揭露银行卡、席位保留。|避免早定性，但误把主钩子也一起删掉。|
|结尾|保留马木没钱/手机没电的轻收束。|轻喜有效，但盖过了“明天中午前保留位置”这个更重要的后续压力。|

## 三、第二版修对的方法

### 1. 误会变得更体面

第二版用“家属配合”“不能跟前台说找谁”“三楼登记”“不方便透露情况”等词，让马雯雯先误以为是某种公职/治安事件，再滑向会所误会。

这比直接写“不正经”“扫黄现场”更可取。后续正文生成应保留这种方法：

```text
让人物用含混、体面、半误解的词维持错误判断，而不是把低俗联想直接说破。
```

### 2. 云水汇正规化是正确方向

平板、耳麦、名单、立牌、登记、预约贵宾这些元素，比“阴暗会所”更能制造错位。

后续生成要继续执行：

```text
可疑场所越正规，误会和不安越强。
```

### 3. 马木不能靠连续金句统治场面

第二版明显减少了马木后段输出，这是对第一版问题的回应。这个方向是对的。

但正确做法不是让马木退场或让关键冲突消失，而是让他更多通过动作、挡人、拽人、让对方确认、打断流程来解决问题。

## 四、第二版新增的问题

### 1. 把概括式幽默换成了更花的概括

“热闹得很努力”本来需要改成饭桌动作、停杯、没人接话等现场反应。第二版改成更密集的旁白吐槽和文气句，仍然没有解决“幽默不在场内发生”的问题。

流程修正：删除概括式幽默时，不能用更漂亮的旁白替换，必须优先改成动作、物件、流程或关系错位。

### 2. 减法误伤了承压场面

第一版后段虽然马木金句过密，但它保留了陈科嘴硬、陈艾艾追问、银行卡、席位保留、工作人员维持流程等核心承压点。第二版把这些大幅压缩后，章节最重要的“从误会到不舒服”的过渡变弱。

流程修正：二次改写可以删金句，但不能删掉让人物暴露真实压力的场面节点。

### 3. 陈艾艾人物弧线被重置

第二次总结要求陈艾艾从“慌、羞、火、控”推进。第二版把她写得过于镇定、强势、江湖气，导致她不再像一个被父亲反复消耗的女儿。

流程修正：支持角色不能因为需要“爽快处理”而丢失情绪弧线。她可以最后变成处理者，但不能一出场就是处理者。

### 4. 主钩子被轻收束盖过

第二次总结明确建议结尾主钩子落在“明天中午前保留位置”。第二版保留了马木没钱/打车的轻喜收束，但缺少更强的后续压力。

流程修正：每章只能有一个主钩子。轻喜收束可以存在，但不能盖过主钩子。

## 五、第二次流程修正要点

1. 二次改写前必须先列出本章“保留、压缩、删除、延后”四类内容。
2. 每场只保留一个主要笑点，但必须保留一个承压动作或承压物件。
3. 角色台词要有胜率控制：关键角色不能每次开口都赢；但删台词时要保留行动功能。
4. 信息披露要延后，但主钩子不能被一起删掉。
5. 误会要体面、含混、双语境成立，不要直白低俗，也不要把误会写成主角降智。
6. 概括式幽默必须转成动作、物件、流程错位或关系反应，不得替换成更漂亮的旁白。
7. 支持角色必须有情绪弧线，不能从头到尾只执行剧情功能。
8. 高风险场面不能被一句“过了一会儿”跳过；如果要压缩，也要保留冲突转折。

## 六、应回写到正文生成流程的内容

本轮已经满足正式修改正文生成 skill 的条件：同类问题在原始草稿、第一版、第二版中反复出现，包括概括式幽默、角色过强、信息边界过早或过度、场面承压点不稳、结尾主钩子不聚焦。

本轮应优先修改 `generate-chapter-prose-draft`，暂不改上游细纲或执行包。原因是：当前问题主要出现在正文写作和二次改写阶段，而不是剧情材料不足。
