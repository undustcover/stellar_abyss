# 第二章第一次修正对比报告

## 一、评估口径修正

上一版评估中将第二章判为“测试通过”，这个判断过宽。

更准确的结论应改为：

```text
链路产物完整，但正文不合格。
```

它确实完成了细纲和执行包里的事实点：身份证夹、纸条、云水汇、陈艾艾、陈科、三万、锁位置、马木带人出来。

但它没有完成第二章真正的阅读任务：

```text
让马雯雯在错误方向上越走越坚定，
让陈艾艾的真实焦急与马雯雯的会所误会持续错频，
让读者先笑，再隐约感到不对。
```

原稿的问题不是“缺信息”，而是“信息都到了，戏没立住”。

## 三、核心差异

|维度|AI原草稿|作者重修第一版|说明|
|---|---|---|---|
|旁白方式|喜欢直接总结气氛和心理，如“成年人的直觉有时候很烦人”。|更多用现场反应和人物互相接话推进，如家里人听到云水汇后的停杯、停筷、追问。|作者版把“作者判断”改成“场内反应”，人物更像活人。|

## 四、AI原稿的真正问题

### 1. 原稿太像“正确地完成执行包”

原稿把每个执行包要求都交代了：

- 身份证夹；
- 纸条；
- 云水汇门口；
- 陈艾艾；
- 三万；
- 资格；
- 锁位置；
- 马木带陈科出来。

但它经常停在“把信息摆到位”，没有把信息变成场内压力。

比如陈艾艾说“我爸这个人特别要面子……银行卡都带了”，这句话信息很全，但不像人在急。作者版改成“洗头、换皮鞋、银行卡塞袜子”，信息没少，但动作更活，也更好笑。

### 2. 原稿让马雯雯太快接近真相

原稿中出现“反诈门口/会所门口”这种近似评估式判断，等于让叙述层提前知道两套语义。

第二章不能这样。

第二章的喜剧主轴是：

```text
陈艾艾以为她们在聊诈骗；
马雯雯以为她们在聊会所；
两人都觉得对方终于懂了。
```

原稿虽然有这个结构，但不够坚决。作者版更好地让马雯雯继续往错误方向跑。

### 3. 原稿的笑点太“旁白型”

原稿有不少顺滑句子，但缺少强反转。

作者版增强了三类场内笑点：

|类型|例子|作用|
|---|---|---|
|误会升级|“我爸也在里面”后，马雯雯理解成陈家也卷入会所问题。|让错误方向升级。|
|现场错位|前台问预约，马雯雯回避登记、说自己是路人。|让云水汇流程参与喜剧。|
|嘴硬碰撞|陈科把钱改叫诚意金、位置、学习机会。|让陈科成为喜剧人物，而不是被解释的人。|

### 4. 原稿过早解释陈科的精神困境

“晚年最后一次证明自己没被时代丢下的机会”这个判断是对的，但太早、太完整。

第2章应该让读者先看到：

```text
陈科嘴硬、端着、好面子、被话术哄得很受用。
```

至于他为什么会被这种话术打中，可以后面再慢慢翻出来。

### 5. 原稿没有让云水汇流程真正施压

原稿写了门口和标识，但流程压力不够。

作者版加入了：

- 访客登记；
- 三楼不方便外人进入；
- 意向确认；
- 工作人员拦门；
- 银行卡需要本人确认；
- 贵宾席位保留到明天中午。

这些都让云水汇不只是“背景”，而是一个会说话、会挡人、会递台阶的系统。

|修正方向|具体效果|可保留为后续修正原则吗|

## 六、作者重修第一版仍未达最终版的问题

|问题|表现|下一轮修正方向|
|---|---|---|
|仍有少量“想要搞笑但偏尬”的句子|批注版已指出“热闹得很努力”偏尬。|删掉或改成动作，不要硬落一句评价。|
|章节标号破坏正文沉浸|正文中保留 `### （一）` 等段落编号。|最终版删除编号，用自然转场。|
|马雯雯误会有时偏低俗边缘|“扫黄现场”“服务业”等方向有效但要收住。|保留错误方向，不展开猎奇画面。|
|马木说“骗钱的”可能过早点破|第二章仍应以会所误会为主，反诈层可露但不宜被完全确认。|可改成更粗糙、更含混的说法，如“这地方就靠人自己往里坐”。|
|马木怼人过密，可能显得太强|PPT模板、银行卡、下辈子等连续有效。|保留一两个强点，避免一口气把马木写成全场唯一清醒的人。|
|章末已有方向但还可更准|三楼灯光、第一场有效；但结尾还可以更贴第三章入口。|落在“明天中午保留席位”或“陈科回头看门口”这类后续压力上。|

## 七、第一次修正结论

本轮不应急着改 skill。

第一步应先修正我们对“合格”的判断标准：

```text
完成信息点 ≠ 正文合格。
误会成立 ≠ 喜剧成立。
信息边界安全 ≠ 读者体验达标。
```

第二章正文合格的最低标准应改为：

1. 马雯雯必须在错误方向上持续推进，而不是快速靠近诈骗真相；
2. 陈艾艾必须像急疯的女儿，不是剧情信息接口；
3. 陈科必须先滑稽、嘴硬、体面，不能过早被解释成悲剧老人；
4. 云水汇必须作为一个“流程系统”参与场景，而不是只当门口背景；
5. 马木必须少解释、多做事，且有效和不靠谱同时存在；
6. 笑点必须来自对话错位、流程阻碍、人物死撑，而不是旁白比喻。

|优先级|修正对象|修正动作|说明|
|---|---|---|---|
|P0|第二章评估口径|将旧评估结论从“测试通过”改为“链路完整但正文不合格”。|避免继续带着错误判断进入第三章。|
|P0|正文 skill 观察项|新增观察：正文是否把执行包信息转化为场内压力，而不是只完成信息点。|先作为观察，不立刻改 skill。|
|P0|第二章下一版正文|以作者重修第一版为基础继续修，不回到AI原稿。|作者版方向更接近正确。|
|P1|高风险错频场景|后续如果继续重写，优先单独处理陈艾艾门口对话。|这里是第二章喜剧核心。|
|P1|云水汇设定|记录外层流程：登记、三楼活动、意向确认、席位保留、银行卡确认。|后续可沉淀为设定卡，但不要解释能力机制。|

## 九、下一步建议

不要马上进入第三章。

建议下一步先做：

```text
第二章第二次问题总结
```

重点不是全面重写，而是检查作者第一版仍未解决的几个点：

1. 哪些句子仍然像“想搞笑”；
2. 马木哪里过强；
3. 哪些地方过早点破“骗钱”；
4. 章末应该落在什么钩子上；
5. 云水汇外层流程哪些能保留为系统资料，哪些只是本章临时表现。

然后再进入第二版修正。
