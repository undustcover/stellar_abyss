# Draft-to-Public 历史分支（目前不可用）

本目录保存已经停止使用的 `Draft → Public` 二次改写分支。

## 状态

> **历史分支，目前不可用。**

该分支试图通过审美规则、表达重构方法、去 AI 化门禁和二次 LLM 改写，将 Draft 转换为 Public。实际测试表明，这套方法容易把小说创作变成规则执行，使人物、叙事节奏和语言失去自然变化，因此停止继续开发和运行。

目录中的 Skill、Prompt、方法库、测试正文、评估报告和 Public 产物仅用于回顾失败过程，不得作为当前正文生成的默认输入，不得自动回灌 Draft 生成链路。

## 当前方向

正文只生成一次。后续工作回到 Draft 的源头生成问题，以“把故事讲得让读者听得懂、愿意听”为目标，重新讨论创作 Prompt 和最小输入。

## 归档内容

- `技能`：二次改写 Skill、通用修订规则及 Public 回灌 Draft 的规则。
- `设计资料`：作者审美链、文风建模、盲写 Public 和相关模板。
- `系统资料`：作者审美决策库、表达重构方法库、去 AI 化讨论及正文修订需求。
- `测试资料`：第一至第四轮 Public、修订、对比和评估材料。
- `正式Public产物`：该分支生成并进入系统目录的 Public 正文。

如未来重新研究二次改写，应另建新分支，不直接恢复本目录为 active。
