# T5 局部复验失败与候选层回退 001

## 一、阶段结论

T5 首个真实失败片段复验失败。

系统将 M03、M04、M07、M02 四项结构候选共同用于局部重构，并在内部反审计中误判为通过；作者实际评级为 D。该结果证明：四项候选目前最多约束说明、动作、认知转向和关系任务的结构，却不足以在第三章真实场景中生成有性格、有灵魂且具有轻喜剧和笑话的正文。

不得把失败解释成个别词句未口语化，也不得继续生成 R2 修补。

## 二、作者直接证据

1. 最先显露系统安排的位置是：

   > 马雯雯觉得这句没毛病，就是没答到地方。

2. 整段不够自然。
3. 没有轻喜剧底色，更没有笑话。
4. 人物性格没有进入正文，实际效果是几个机器人在对话，谈不上人物，也没有灵魂。

完整原话逐字保存在来源测试卡。

## 三、直接可观察问题

### 3.1 POV 旁白继续替系统判卷

“没毛病，就是没答到地方”不是马雯雯在现场中形成的独特感受，而是系统替读者解释上一组问答的功能。它与失败原文中的“分类显然更合理”属于同一问题，只是换成更口语的措辞。

### 3.2 对话仍按任务字段排列

复验依次填写：次数、来源、小刘身份、老师职责、调理过程、记录表。虽然删掉书面状态标签，人物仍只负责提问、回答、插问和拒绝，任何同类人物都可以替换进来。

### 3.3 M02 没有真正进入人物生成

系统声称陈科在维护自主性和所有权，但正文只留下“我自己会走”“我的东西别翻”两项通用防御。陈科的体面、自我理解和被服务后的满足没有形成只属于他的说话过程；陈艾艾也只有盘问功能；马木只有一句工具性插问。

### 3.4 轻喜剧被误判为“存在错位”

“自己会走”和“会所一般不发这个”被系统当作现实错位，但作者没有读到轻喜剧，更没有读到笑话。这证明喜剧不能由内部标签宣布成立，必须在人物持续行动、误解和关系碰撞中产生可感知效果。

### 3.5 四项候选共同合规不等于正文成立

M03、M04、M07、M02 可以帮助排除部分说明书化、装饰动作和无触发转向，却没有提供：

- 当前人物可调用的具体语言材料；
- 人物欲望、情绪、关系和自我理解如何连续进入对白；
- 轻喜剧如何形成、升级和落点；
- 如何识别“所有句子都合规，但仍是机器人问答”。

## 五、回退任务

下一步不再直接重写第三章片段，而是返回候选与生成设计层：

1. 从已确认的人物材料和作者认可正文中，区分“人物资料”与“人物实际语言材料”；
2. 建立可检验的“人物欲望／情绪／关系旧账／自我理解如何进入连续对白”候选，不用口癖或金句代替人物；
3. 单独建立轻喜剧与笑话的实例级验证，不能再把“现实错位”自动判成喜剧成立；
4. 新增“姓名可替换测试”“机器人任务问答测试”“POV 代系统判卷测试”和“作者是否实际读到笑话”门禁；
5. 完成新的非正史测试并获作者通过后，才能重新选取第三章真实失败片段复验；
6. 在新的真实片段复验通过前，禁止整章 Draft→Public。

## 六、禁止错误归因

- 不得归纳成“作者只要更口语的对白”。
- 不得把“马雯雯觉得”改成一个更俏皮的旁白后重交。
- 不得给每个人增加口癖、动作、冷刺或金句来伪造性格。
- 不得把普通问答打散、加停顿或删标签后声称人物已经鲜活。
- 不得声称存在轻喜剧，除非作者在实际文字中读到轻喜剧或笑话。
- 不得要求作者改写该失败片段来替系统补齐人物和喜剧生成能力。
