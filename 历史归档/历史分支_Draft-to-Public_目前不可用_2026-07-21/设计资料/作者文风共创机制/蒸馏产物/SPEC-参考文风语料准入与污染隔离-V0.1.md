# 参考文风语料准入与污染隔离 V0.1

## 1. 本轮准入结论

|字段|登记值|

|语料 ID|CORPUS-LSR-QIDIAN-001|
|文件|`设计流程与方案/作者文风共创机制/示例文风/捞尸人_起点.md`|

|准入状态|`admitted_limited_scope`|

|不可做|宣称代表作者全部作品；推断样本未覆盖的感情线、群战、都市职场、长线高潮等场景|

该样本具有足够的连续篇幅、人物数量和场景变化，可以支持首轮蒸馏。其不足不是“字数太少而完全不能分析”，而是覆盖域有限。因此本轮采用“可以蒸馏，但结论带适用域”的处理。

## 2. 准入检查

- [x] 来源路径明确，由作者主动指定。
- [x] 正文与当前小说 Draft/Public 分离。
- [x] 正文为连续章节，不是模型挑选的漂亮句集。
- [x] 包含日常、对白、说明、动作、恐怖、民俗仪式等多类文本。
- [x] 能划分出未参与首轮归纳的留出片段。
- [x] 原文只进入语料层和证据层，不进入运行时提示库。
- [ ] 尚无同作者其他作品或本作后续章节可用于跨作品、跨卷验证。
- [ ] 来源版本、抓取时间和正式出版校勘状态未登记；不影响本轮风格观察，但影响严格版本追踪。

## 3. 污染隔离

### 3.1 四层隔离

1. `raw`：保存完整参考正文，只允许定位证据。
2. `evidence`：仅保存位置、短摘记、现象和反例，不保存可拼接原句库。
3. `distilled`：保存深层机制、适用条件、失败边界和置信度。
4. `runtime`：本轮不创建。只有经作者实例校准并正式化的当前小说规则可进入。

### 3.2 禁止迁移

- 参考作品专名、人物、称谓、方言词、民俗行话和物件组合；
- 可识别的原句、排比、章尾包袱或恐怖意象组合；
- “水、尸、铃铛、香炉、烟、吃席”等表面元素；
- 单纯复刻短句、断行、拟声词、方言密度或先笑后惊的固定节拍。

### 3.3 可进入作者校准的内容

只有抽离具体名词与句序后的机制可以进入下一阶段，例如：让一个先前承担日常意义的对象在后续事件中发生可见变化，从而同时推进信息和情绪。该机制仍只是 `observed`，不能直接成为当前小说规则。

## 4. 版本与扩充条件

新增语料时必须重新登记：来源、章节范围、是否连续、是否被模型改写、覆盖的场景类型。新增样本与 V0.1 冲突时，优先降低结论置信度，不得用新样本强行解释旧证据。
