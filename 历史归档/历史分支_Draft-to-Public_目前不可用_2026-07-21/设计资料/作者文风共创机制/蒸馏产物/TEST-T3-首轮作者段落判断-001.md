# T3 首轮作者段落判断

本轮有两个互不连续的小实例。请分别判断；不需要比较哪一个“更正确”，也不需要解释文风理论。

## 实例一

### 最小上下文

暴雨后的第二天，许宁去物业取被暂扣的电动车。值班员不肯把车交给她。

### 测试原文

许宁把取车单递进窗口，值班员没接，先指了指车筐里那块鼓起来的充电宝。

“这个不能进电梯。”

“它昨天还好好的。”

“昨晚泡过水。”值班员把桌下的透明箱拖出来，里面已经放着三块同样鼓起的电池，“进水以后不一定马上冒烟，先发热。你那辆车刚才在库里报了两次温。”

许宁的手还搭在窗口上。

值班员推来一只铁夹子：“钥匙给我。你先别碰车。”

### 请作者判断

1. `A 自然` / `B 自然但当前小说不要` / `C 局部有问题` / `D 一眼 AI` / `E 需要更多上下文` D
2. 最先出戏的位置在哪里？对话依然是AI式的说明，解释；依然是完整结构的说明文。
3. 这段是否像小说，是否适合当前小说整体读感？不像，完全不合适
4. 可选：你会怎样改一处？
{{作者改写/}}
许宁把取车单递进窗口，值班员正在刷视频的手根本没停，抬眼看了下取车单，接着对着车筐昂了昂头。

“怎么了？”许宁问道。

“贴条了。”值班员一只手拿着手机，一只手把桌下的透明箱拖出来，“2块鼓泡，一块起火了，不能让你取，万一进电梯着火了算谁的？“

许宁的手还搭在窗口上。

值班员又玩起了手机。
{{/作者改写}}
## 实例二

### 最小上下文

方澄替同事送一箱样品去商场库房。仓管看完送货单，却让她把箱子重新封好。

### 测试原文

仓管把送货单压在箱盖上：“这批今天进不了库。”

方澄看了一眼时间：“预约单、件数、封签都在，还差什么？”

“不是差东西。”仓管抬头看向卸货口。门外停着一辆没有商场标识的冷藏车，车门敞着，两个穿便装的人正往里搬空周转箱，“库里刚断过一次电。温度记录没补齐以前，谁的货进去，损耗就先算谁的。”

方澄把已经拆开的胶带按了回去。

仓管抽走送货单，声音低下来：“车别熄火。你要是能联系上采购，现在就打。”

### 请作者判断

1. `A 自然` / `B 自然但当前小说不要` / `C 局部有问题` / `D 一眼 AI` / `E 需要更多上下文` D
2. 最先出戏的位置在哪里？对话依然是AI式的说明，解释；依然是完整结构的说明文。
3. 这段是否像小说，是否适合当前小说整体读感？不像，完全不合适
4. 可选：你会怎样改一处？
{{作者改写/}}
仓管没接送货单，“停电了，进不了”

方澄看了一眼时间：“和我有什么关系，不入库我交不了差。”

“你交不了差我就能交了？”仓管向门口撸了撸嘴。门外停着一辆没有商场标识的冷藏车，车门敞着，两个穿便装的人正往里搬空周转箱，“赶紧走，别浪费大家时间”

方澄把已经拆开的胶带按了回去。

仓管看着不知所措的方澄，叹了口气，心软的提示到：“打电话啊，要么问人，要么摇人，你愣着能解决问题？”
{{/作者改写}}
