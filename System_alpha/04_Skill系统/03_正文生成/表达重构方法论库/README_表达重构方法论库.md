# 表达重构方法论库运行说明

## 一、目录定位

本库负责把作者审美判断转换为 L2—L4 表达动作，供 Draft→Public 和经授权的局部表达重构使用。

本库不判断作者保护什么；该职责属于作者审美决策库。本库也不保存作者原句、章节专属桥段、固定段落顺序或成品正文。

## 二、文件职责

|文件|职责|
|---|---|
|`LIB-EXPRESSION-P0_表达重构方法论库.md`|正式跨场景表达重构方法|
|`CHANGELOG_表达重构方法论库.md`|条目新增、调整、降级与废弃记录|
|测试目录中的候选方法文件|保存测试来源、作者校准和具体问题证据|

## 三、使用顺序

```text
审美库判断问题与保护价值
→ 片段审计确定 L0—L4
→ 本库选择一个主方法和至多两个辅助方法
→ Public Skill 执行
→ 三类门禁验证
```

## 四、写入边界

正式条目必须：

- 具有识别信号、保留项、步骤、适用与不适用条件；
- 具有失败边界和修改深度检查；
- 能跨至少两个不同功能片段迁移；
- 经作者确认适用边界；
- 不包含作者原句和章节专属成品方案。

## 五、等级

当前条目均为 `L3 作者确认方法`。L3 只表示边界获得作者确认，不表示无条件调用。
