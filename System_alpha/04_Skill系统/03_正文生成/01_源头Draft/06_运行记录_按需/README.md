# 运行记录索引

普通生成保存一份紧凑的统一 Draft 运行日志，但不在本目录创建详细运行实例。本目录只用于高风险章节、正式测试、跨窗口恢复或故障复现时的扩展记录；当前没有详细运行实例。

统一日志接口见 `../02_方案/PLAN-CHAPTER-INTERFACE-001_章节DraftBrief与统一运行日志.md`。详细记录若被启用，必须由统一日志说明原因并引用，不得取代默认紧凑日志。
