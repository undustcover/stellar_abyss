# 方法论索引

本目录保存源头 Draft 样板阶段与作者共创的方法源码。它不是独立 RAG 知识库；正式化后，每种方法应归入对应模块 Skill，由 Skill 通过轻量索引按需读取。

当前规则：

- 没有作者确认的方法时先讨论，不由 LLM 静默建立写作方法；
- 作者确认后记为 `待正文验证`，产物获得作者认可后记为 `已验证`；
- 只保存适用情境、目标效果、方法原理和创作自由，不保存成品句、固定桥段或公式化动作；
- 当前不从旧规则库批量继承，不建立向量 RAG 或 Python 检索。
