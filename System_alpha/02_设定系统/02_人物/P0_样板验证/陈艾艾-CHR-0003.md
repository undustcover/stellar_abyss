# 陈艾艾

## 人物摘要

陈艾艾，女，新TK市居民，陈科的女儿。

她在单亲家庭中长大，陈科从小把她当男孩养，导致她形成了偏男性化、独立、强势、遇事先处理问题的性格。她和陈科都硬，都不太会好好说软话，父女关系长期处在顶撞、压火、互不服气的状态。

她是陈科事件里最早感到现实危险的人。她不承担喜剧制造功能，而是用正常、急迫、现实的追问把陈科的荒唐托出来。她关心的不是格局、资格或位置，而是父亲到底有没有转钱、签字、交出身份信息，以及还能不能止损。

陈科并不是不爱她。相反，在多数现实问题上，陈科会听陈艾艾的，也会暗地里照顾她、让着她。但唯独诈骗和“被尊重感”相关的事情上，陈科会突然变得固执，两人进入水火不容的状态。父女矛盾的底色不是不爱，而是都太强势、都不肯先低头。

陈艾艾对马木有信任，正是这种信任把马木带入陈科事件。事件反噬后，她被父亲、证据和现实流程夹住，情感上知道马木是在帮忙，现实上却难以完全站到马木一侧。
