# 马雯雯

## 人物摘要

马雯雯，女，新TK市职场人，马木的堂妹。

她目前的最小信息定位是：以正常年轻亲属和职场压力承受者的视角观察马木、陈科、陈灏、张富贵等人的荒唐处境。她起初轻视马木，把他看成不靠谱、混得差、爱吹牛、疑似混会所的堂哥；随着陈科事件、证据链和现实反噬展开，她逐渐意识到自己看到的荒唐并不等于真相。

马雯雯的语言承担误读和即时反应功能，但她不是段子手。她的吐槽来自亲属尴尬、职场疲惫和信息错位。

## 人物时间轴

### TBD-YYYY-MM｜马木亲属误读期

马雯雯把马木看成家族里不靠谱的中年堂哥。她对马木的认知来自家族气氛、马木外在形象、电话碎片和云水汇误会。

### TBD-YYYY-MM｜陈科事件旁证期

马雯雯亲眼见到马木、陈科、陈艾艾、陈灏等人围绕云水汇和陈科资金问题纠缠。她的误读制造喜剧，也使她成为后续正式询问中的事实旁证。

### TBD-YYYY-MM｜现实反噬认知转变期

派出所询问和证据链重组后，马雯雯意识到马木的荒唐和热心可以被现实规则翻译成参与痕迹。她开始重新理解马木。
