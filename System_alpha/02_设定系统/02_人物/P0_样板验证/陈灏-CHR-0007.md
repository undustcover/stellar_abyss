# 陈灏

## 人物摘要

陈灏，男，新TK市年轻男性，具体家庭背景待补充，目前可确认其表层位置是有钱、轻浮、爱挑刺的富二代式人物。

他不是真正的反诈专家，也不负责解释异常世界观。他的有效判断来自两处：一是从小见过真正的高端场合，因此能识别云水汇过度努力的假高端；二是存在轻微异常感知，能捕捉场子、设备、节奏和话术里“不对劲”的地方，但他自己也未必知道为什么。

陈灏的最小人物卡只记录这一阶段的世界事实：他说话欠、观察准、不负责完整解释，常让别人以为他在胡说。
