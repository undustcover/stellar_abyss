# 马木

## 人物摘要

马木，男，40岁，新TK市自由职业者，居住在十平米出租屋中。

他在现实社会中没有稳定体面的位置，却长期承担亲友和熟人之间的临时问题处理者角色。别人常把他看成不务正业、嘴贫、不稳定的人，但在具体事务中，他能快速看出责任边界、话术结构、人情压力和现实风险。

马木的核心矛盾不是无能，而是他的判断力足以看穿局部问题，却不足以抵抗更高层级的结构性陷阱。他习惯先处理眼前的火，尤其难以拒绝熟人求助。这种行为模式让他在多次现实事件中短暂有效，也使他被卷入无法凭善意解释清楚的责任链条。

在完整时间线中，马木由新TK市的现实边缘人，逐步成为《六合集市》APP拥有者与《六合集市》第一任主任。

|装腔或失误后的反应|说过头后会咳嗽、改口、补一句“这个可以忽略”之类的话；越找补越容易显得可疑|

|与弱势/需要保护对象说话方式|嘴上不软，行动会先止损；不擅长温柔安抚，常用粗糙判断让对方先离开危险处境|
|不会使用的表达|通常不会使用文学化、导师式、心理咨询式、系统说明式语言；不会在不知道异常体系时使用超凡概念解释现实问题|
|语言边界形成原因|长期在现实麻烦、人情求助和责任推诿中救火，使他相信具体后果比漂亮说法可靠；落魄和旧事让他不愿把关心说得体面|
