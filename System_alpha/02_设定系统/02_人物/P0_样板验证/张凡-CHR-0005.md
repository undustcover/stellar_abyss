# 张凡

## 人物摘要

张凡，男，张富贵的儿子，新TK市居民，具体年龄待补充。

他长期处在严重封闭状态中，不是普通不爱说话，也不是单纯叛逆。小满是他与外界保持稳定连接的少数对象之一。张凡的存在不靠语言推动事件，他的沉默、不开门、不回应和通过小满表达需求，构成张富贵线真正的现实重量。

在小满离开之前，外人容易把张凡家的情况误读成普通父子沟通困难和家庭喜剧；小满离开之后，张凡的封闭让前面的笑点全部显出真实代价。
