# 苏芳菲事件

## 事件摘要

苏芳菲事件是六合集市能力从“效果夸张的日用品”升级到“教程与手搓装置”的主线事件。马木通过《六合集市》接触【深空神域使团】相关教程，并用简笔画教程和手搓纸箱装置制造出足以影响意识、感知和虚荣阈值的体验。苏芳菲在该体验中接触到远超现实消费和身份炫耀的宇宙神豪式幻象，虚荣阈值被撑爆。

马木不会把这理解为高维意识干涉，只会把纸箱装置解释为劣质VR、催眠、视觉暂留、心理暗示或廉价手工黑科技。世界真相层面，本事件证明APP可以让用户手搓出干涉意识和感知的神器，且不直接破坏物理日常。
