## 5. 当前有效结论

截至本次盘点，可直接确认：

1. `System_alpha` 是唯一正式系统目录；
2. 系统采用用户与 Codex 共同迭代的开发模式；
3. 世界事实、叙事安排、人物意图、正文执行和测试证据必须分层；
4. Draft-to-Public 二次改写分支已经停止且不可作为当前输入；
5. 当前正文只生成一次，质量问题优先从生成源头、输入最小化和 Prompt 设计解决；
6. 每篇正文经用户修改后都应做方法归因，但具体句子、桥段和成品解法不能未经确认直接沉淀为通用规则；
7. 正式 Skill 源码最终归入 `System_alpha/04_Skill系统/`，但当前 Draft Skill 仍在设计中，不安装、不迁移。
