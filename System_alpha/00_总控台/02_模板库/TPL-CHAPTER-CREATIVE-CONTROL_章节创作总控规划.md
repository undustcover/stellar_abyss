# 章节创作总控规划模板

章节创作总控规划是总-分-总链路的前置总规划文件。

它只负责统一本章各产物和 skill 的分工、接口、互补关系、引用关系、冲突优先级和 token 边界。它不是剧情大纲，不是正文总包，不是场景演出契约，也不是章节执行包。

导演层可复用文本质量规则属于正文草稿生成 Skill 的 reference。本规划只登记该规则库在本章链路中的读取关系和禁止复制边界，不复制规则全文，不替代场景演出契约，也不替正文草稿生成 Skill 执行去 AI 化或文风判断。

## 文件命名

```text
CTRL-{单元编号}-{章节编号}_{章节名}_章节创作总控规划.md
```

示例：

```text
CTRL-P0-C03_这事好像比会所还怪_章节创作总控规划.md
```

## Token 边界

|规则|说明|
|---|---|
|禁止线性复述|后续产物只引用本规划的分工和接口，不复制全文|
|禁止总包化|本规划不得写完整剧情、完整场景、完整人物资料|
|保留追溯能力|需要追溯作者原意时读取共创讨论记录|
|导演层规则边界|只登记导演层文本质量规则的读取关系，不复制规则全文，不把本规划扩写为文本质量总包|

## 越权排除检查

|检查项|通过标准|
|---|---|
|是否写成剧情大纲|没有完整剧情、场景顺序、事件推进或场景起止状态|
|是否写成场景演出契约|没有具体句式、对白自然化细则、去 AI 化操作清单或文字演出规则全文|
|是否写成人物意图结果|没有人物意图轨道、语言/行为意图、认知轨道或 `CONTINUE/ASK_USER/RESUME`|
|是否写成章节执行包|没有场景权重、生成顺序、转场表、风险路由或字数调度|
|是否写成正文草稿|没有正文成品句、完整对白、完整 punchline、段落顺序或结尾句|
|是否复制导演层规则|没有复制导演层通用文本质量规则全文，只登记读取关系和禁止复制边界|
|是否保持接口总线|只规定分工、互补、引用、冲突优先级、ASK_USER 和 token 边界|
