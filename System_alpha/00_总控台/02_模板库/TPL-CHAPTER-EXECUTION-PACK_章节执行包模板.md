# 章节执行包模板

章节执行包用于在总-分-总正文生成链路中提供调度索引、场景权重、生成顺序、转场状态、风险路由和人物意图覆盖状态。

它不是章节总包，不复述上游产物，不替代章节细纲、场景演出契约、人物意图结果或正文总导演。

执行包只把已确认产物转成“正文生成时如何调度”的索引语言：路径、ID、场景编号、权重范围、状态标签、风险标签、处理层和回流建议。它不得把上游内容重新写成总摘要，也不得把调度语言写成正文方案。

## 文件命名

```text
PACK-{单元编号}-{章节编号}_{章节名}_章节执行包.md
```

示例：

```text
PACK-P0-C03_这事好像比会所还怪_章节执行包.md
```

## 越权排除检查

|检查项|通过标准|
|---|---|
|是否写成上游内容总包|没有复述章节契约、细纲、场景演出契约、人物意图结果或人物/事件/设定资料全文|
|是否替代章节细纲|没有新增剧情事实、改写场景顺序、改写场景起止状态或补写事件推进|
|是否替代场景演出契约|没有复制完整文字演出规则、通用去 AI 化规则或对白自然化规则全文|
|是否替代人物意图结果|没有生成完整人物意图轨道、语言意图、行为意图、动机推导或 `CONTINUE/ASK_USER/RESUME`|
|是否替代正文草稿|没有正文成品句、完整对白、完整 punchline、具体段落顺序、结尾句或准正文桥段|
|是否只做风险路由|高风险项只写来源、处理层、是否 ASK_USER 和回流建议，不自行修复业务冲突|
|是否保持临时产物性质|没有写入长期世界事实，没有要求回写人物卡、事件卡或设定卡|
