# Chapter Detail Outline Template Pointer

The formal chapter detail outline template is the source of truth:

```text
System_alpha/00_总控台/02_模板库/TPL-NAR-DETAIL-OUTLINE_章节细纲模板.md
```

When generating a `type: detail_outline` card, use that formal template's YAML fields, heading order, table names, and boundary notes exactly.

Do not use older local test structures that contain:

- `章节文风模式`
- `场景文风分配`
- `场景导演目标`
- `人物语言调用`
- `反杀材料链`
- standalone `对话重点`

Use the formal replacements:

|Old/local wording|Formal wording|
|---|---|
|人物语言调用 / 人物语言边界索引|人物边界索引|
|场景列表|场景总表|
|反杀材料链|可回收材料链|
|对话意图与话题|对话意图与话题范围|

The outline must remain the plot-skeleton layer: scene order, action movement, information release, cognition gap, pressure points, reader-experience traceability, and intent-control risk preparation.

Route prose style, anti-AI rules, dialogue naturalness, sentence rhythm, and key-scene written performance to `co-create-scene-performance-contract` or the final prose director layer.
