# 1. 核心设定

- 轮回观察者组织叛逃人员。
- 高维异能者。
- 表层身份是新TK市第一中学高二八班学生。
- 暗线身份是活过极长时间、携带轮回观察经验和组织秘密的叛逃者。
- 与徐乂共同承担后续卷的世界观揭示线。

# 2. 人物档案

- 性别：女
- 年龄：外表17岁，真实年龄约1500岁
- 表象身份：新TK市第一中学，高二八班学生
- 暗线身份：轮回观察者组织叛逃人员
- 居住地：新TK市
- 当前状态：隐藏身份，避免被组织重新定位

# 3. 稳定特征

## 3.1 表层状态

林薇在现实生活中必须表现得像普通高中生。她不能过度暴露知识、判断和经验，否则会引来组织和异常势力注意。

她与徐乂同在新TK市第一中学，但早期不必直接相识或同框。两人可以各自在不同视角中感知世界裂缝。

## 3.2 暗线状态

林薇知道世界不是单线运行，也知道某些事件存在重复、偏移和被观察的痕迹。

她不是普通“重生者”，而是轮回观察者体系中的叛逃者。她的知识来自漫长观察、组织训练和反复见证，而不是一次简单的未来记忆。

## 3.3 核心矛盾

林薇知道得太多，但不能说太多。

她既要避免组织追索，又要在关键节点干预世界走向；既知道徐乂重要，又不能过早把徐乂推入暴露状态。

# 4. 能力与限制

## 4.1 能力倾向

- 对时间偏移、现实错位和重复结构敏感。
- 能识别某些“本不该这样发生”的事件。
- 可能保留轮回观察者组织的观察技术、判断方法或隐蔽规则。
- 具备高维异能者特征，但具体能力名称与边界待补。

## 4.2 限制

- 不能随意解释世界真相，否则会改变被观察对象和世界线。
- 不能轻易暴露自己是叛逃者。
- 不能直接替徐乂完成权限觉醒。
- 不能把轮回经验当成万能解法，历史可能已经偏离她熟悉的轨道。

# 5. 性格特征

- 表面冷静、克制、像普通学生。
- 内在极度疲惫，长期承受重复观察和逃亡压力。
- 不轻易信任人，但会对关键人物保持观察。
- 对“命运必然”保持怀疑，因为她见过太多被重复、修剪或改写的结局。

# 6. 核心动机

## 6.1 表层动机

隐藏身份，维持普通生活，避免被轮回观察者组织重新捕获。

## 6.2 深层动机

打破组织对时间线、观察对象和文明命运的控制。

她不只是想活下去，也想证明被观察的人不是样本，不该被组织当作可反复推演的实验对象。

# 7. 人际关系

## 徐乂

徐乂是失落人类帝国权限继承者，也是后续世界线争夺的核心。

林薇可能比徐乂更早意识到他的异常价值，但她不能过早把全部真相告诉他。她与徐乂的关系应在“保护、试探、隐瞒、共担真相”之间推进。

## 马木

马木是第一卷《六合集市》线的核心人物。他的行为会造成幽影之墙消耗和地球坐标暴露风险，也会最终把灾难证据链交给徐乂。

林薇对马木的判断不应简单等同于敌人或救世者。她需要在后续信息中逐渐看清马木悲剧的结构性原因。

## 轮回观察者组织

林薇是叛逃者。组织既是她的知识来源，也是她的追捕者和精神枷锁。

# 8. 角色弧光

## 起点

隐藏在高中生活中的异常人物，察觉时间错位和现实裂缝。

## 介入期

逐步接近徐乂与马木相关事件，但仍控制信息暴露。

## 结盟期

与徐乂共同面对失落人类帝国遗产、幽影之墙、外部文明和轮回观察者的多重压力。

## 反抗期

从逃离组织，转向主动破坏组织对世界线和文明命运的控制。

# 9. 隐藏信息

- 林薇真实年龄约1500岁。
- 她不是普通高中生，而是轮回观察者组织叛逃人员。
- 她知道部分世界线和异常机制，但不一定知道全部宇宙真相。
- 她与徐乂共同承担后续世界观揭示线，但不能替徐乂觉醒。

# 10. 写作边界

- 不把林薇写成全知全能解释器。
- 不让她过早向徐乂或马木讲透世界真相。
- 不把她的长寿写成单纯卖萌反差。
- 不让她的轮回经验完全压倒当前现实，因为世界线正在偏离。
- 林薇性别统一为女。
