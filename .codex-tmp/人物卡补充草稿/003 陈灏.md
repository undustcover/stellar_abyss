# 1. 核心设定

- 异能者：[[机械之心]]
- 表面身份：富二代、创维科技老板
- 暗线身份：异能者，[[火炬之光]]成员
- 第一卷功能：脚本嗅觉、异常素材记录者、读者奖励机
- 他不是来解释真相的人，而是让读者感觉“这场子不对劲，而且很好拍”的人。

# 2. 人物档案

- 性别：男
- 年龄：25岁
- 表象身份：创维科技老板、富二代、内容/科技圈玩家
- 暗线身份：异能者，[[火炬之光]]成员
- 居住地：新TK市
- 关键能力：[[机械之心]]

# 3. 稳定特征

## 3.1 表层气质

陈灏看起来像有钱、有闲、有设备、有渠道的人。他对新奇事件、奇怪人物和有拍摄价值的现场格外敏感。

他的表层动机可以是找素材、做内容、记录社会怪象或科技奇闻。这个表层动机让他自然出现在分享局、异常现场和马木相关事件边缘。

## 3.2 暗线气质

陈灏不是普通看热闹的富二代。他的[[机械之心]]让他对结构、设备、系统、机械逻辑和异常痕迹有特殊敏感度。

他能感到“这不是普通故障”，但未必能完整解释高维、灵能、六合集市或幽影之墙。

## 3.3 核心矛盾

陈灏介于旁观者和介入者之间。

他习惯先记录、先分析、先确认素材价值，但当异常真正伤人时，他不能永远躲在镜头和设备后面。

# 4. 能力与限制

## 4.1 机械之心

具体能力边界以[[机械之心]]设定卡为准。

当前人物卡只记录稳定方向：

- 对机械、电子设备、系统结构和技术异常敏感。
- 能捕捉普通人忽略的故障、延迟、数据残影或设备不合理反应。
- 适合承担异常现场记录、证据捕捉、技术侧旁证等功能。

## 4.2 限制

- 不能替马木解释全部真相。
- 不能直接破解六合集市、幽影之墙或帝国权限。
- 机械之心不等于万能黑客能力。
- 他记录到异常，不代表能说服所有现实流程。

# 5. 核心动机

## 5.1 表层动机

寻找值得拍、值得研究、值得传播的异常素材。

## 5.2 深层动机

确认自己感知到的世界裂缝不是幻觉，并找到能够理解这些裂缝的人或组织。

加入[[火炬之光]]后，他的动机从“拍到好东西”逐渐转向“记录、验证并处理异常”。

# 6. 人际关系

## 马木

陈灏在陈科被骗事件中可作为异常旁证。他能感到现场不对劲，但不能把完整机制解释给马木。

马木在早期只会把陈灏当成富二代、拍素材的人或技术圈怪人，而不会立刻理解其异能身份。

## 任战

任战体系造成的异常影响，是陈灏可能捕捉到“不对劲”的对象之一。

陈灏与任战不必早期正面对抗，但他记录到的素材可成为后续追查线索。

## 火炬之光

陈灏是[[火炬之光]]成员。组织为他提供异常认知框架，也限制他不能随意公开所有素材。

## 徐乂与林薇

陈灏与徐乂、林薇可在后续异常调查中形成互补：徐乂是权限核心，林薇是时间/观察线，陈灏是技术记录和设备旁证线。

# 7. 角色弧光

## 起点

有钱、有设备、有嗅觉的异常素材猎人。

## 卷入期

通过分享局、设备异常和马木相关事件，确认新TK市正在发生普通科技无法解释的事情。

## 转向期

从“记录异常”转向“承担异常证据责任”，明白镜头不是中立的安全区。

## 成熟期

成为主角团技术侧和证据侧的重要支撑，但仍受能力边界和组织规则限制。

# 8. 隐藏信息

- 陈灏是异能者，不只是富二代。
- 他属于[[火炬之光]]，但早期不应把组织信息全量暴露。
- 他可以提供异常旁证，但不能替代世界观揭示主线。

# 9. 写作边界

- 不把陈灏写成纯搞笑富二代。
- 不把[[机械之心]]写成万能黑客外挂。
- 不让陈灏过早解释六合集市和幽影之墙真相。
- 他记录到异常不等于现实流程会采信异常。
- 他的价值在于“捕捉痕迹”和“技术旁证”，不是替所有人解决问题。
