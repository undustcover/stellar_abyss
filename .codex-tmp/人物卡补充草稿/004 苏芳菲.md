# 1. 核心设定

- 异能者：[[先视之界]]
- 表面身份：假名媛、富人圈边缘参与者、体面感制造者
- 暗线身份：异能者，马木的头号粉丝
- 第一卷前段功能：分享局里的信任放大器。
- 后续功能：被六合集市体验打穿虚荣阈值后，转向对马木产生强烈崇拜和追随。

# 2. 人物档案

- 性别：女
- 年龄：28岁
- 表象身份：假名媛
- 暗线身份：异能者，马木的头号粉丝
- 居住地：新TK市
- 关键能力：[[先视之界]]

# 3. 稳定特征

## 3.1 表层气质

苏芳菲要高级、体面、轻慢，不能写成脸谱坏女人。

她擅长制造门槛感、稀缺感和被筛选感。她的价值不是直接骗钱，而是让底层人相信“这里不是骗局，这里是门槛”。

## 3.2 暗线气质

苏芳菲是异能者，拥有[[先视之界]]。她对未来、机会、风险或人物潜力有某种异常感知，但具体能力边界待能力卡补充。

她不是单纯拜金女。她真正执着的是被更高层级承认，被看见，被带入“真正的上层叙事”。

## 3.3 核心矛盾

苏芳菲渴望进入更高世界，但她早期误把消费等级、身份包装和圈层门槛当成“更高世界”。

六合集市相关体验会把她的虚荣阈值撑爆，使她意识到现实名媛游戏的尺度过小，从而发生价值坍塌和重组。

# 4. 能力与限制

## 4.1 先视之界

具体能力边界以[[先视之界]]设定卡为准。

当前人物卡只记录稳定方向：

- 与预见、先视、机会判断或未来片段有关。
- 可能让苏芳菲比普通人更敏感地捕捉“谁值得押注”。
- 可能是她后续成为马木头号粉丝的重要原因之一。

## 4.2 限制

- 先视不等于全知。
- 她可能看见趋势、光晕、可能性或片段，但未必理解代价。
- 她的虚荣、身份欲望和自我包装会污染她对先视结果的判断。
- 早期不能让她直接解释六合集市真相。

# 5. 核心动机

## 5.1 表层动机

维持体面形象，进入更高级圈层，获得被上层承认的身份位置。

## 5.2 深层动机

寻找真正值得追随的“更高叙事”。

她早期以为更高叙事是财富、名媛圈、投资局和阶层门槛。苏芳菲事件后，她会意识到马木和六合集市背后的东西远远超过她原本追逐的体面游戏。

# 6. 关键事件定位

## 分享局阶段

苏芳菲在陈科相关分享局中承担信任放大器功能。

她的轻慢和体面感会让陈科这类人相信：自己不是被骗，而是终于被允许靠近某个上层入口。

## 苏芳菲事件

苏芳菲通过【深空神域使团】相关教程和手搓纸箱装置，体验到远超现实消费与身份炫耀的宇宙神豪式幻象。

她的虚荣阈值被撑爆，原本追逐的名媛式体面变得狭小、廉价、可笑。

马木不会理解该体验的高维本质，只会把它解释为劣质VR、催眠、视觉暂留或纸箱万花筒式黑科技。

# 7. 人际关系

## 马木

苏芳菲后续是马木的头号粉丝。

她崇拜马木，不是因为马木表面体面，而是因为马木误打误撞打开了她见过的最大尺度世界。

马木对苏芳菲的理解会滞后。他可能把她当成假名媛、麻烦人物、体验者或夸张粉丝，而不会立刻理解她的异能和崇拜结构。

## 陈科

在分享局阶段，苏芳菲可成为陈科身份幻觉的放大器。她不一定需要直接加害陈科，但她的存在会强化“这里是门槛，不是骗局”的错觉。

## 陈灏

陈灏可能在分享局或异常事件中记录到苏芳菲的状态变化。两人一个偏体面/先视，一个偏技术/记录，可形成互相看不顺眼但都能提供旁证的关系。

# 8. 角色弧光

## 起点

假名媛、体面感制造者、分享局信任放大器。

## 冲击期

通过六合集市相关体验，见到远超现实名利游戏的尺度，虚荣阈值被撑爆。

## 重组期

从追逐圈层承认，转向追随马木和六合集市背后的更大世界。

## 成熟期

她需要从“崇拜马木”转向真正理解马木、异常体系和自己能力的代价。

# 9. 隐藏信息

- 苏芳菲是异能者，不只是假名媛。
- 她的[[先视之界]]可能让她提前感到马木的特殊价值。
- 她早期作为分享局信任放大器，不等于她完全理解任战体系和六合集市真相。
- 她后续对马木的粉丝化，来自价值尺度被彻底打穿后的重组。

# 10. 写作边界

- 不把苏芳菲写成脸谱坏女人。
- 不把她的拜金写成单纯爱钱；核心是身份、门槛和被承认。
- 不让她过早理解六合集市和高维意识干涉真相。
- 不把“马木头号粉丝”写成无脑追星，要有先视、尺度冲击和价值重组支撑。
- `苏芳菲`为当前统一用名，避免与`苏芬菲`混用。
