## 全章文字演出基调

本节只写本章相对导演层通用规则的差异、加强或例外。S1-S2 是本次测试范围，S3-S5 待扩测。

|项目|规则|禁止写法|
|---|---|---|
|句式节奏|S1-S2 允许短句、半截反应和生活化转念；重点是粗糙生活脑回路，不追求漂亮收束|每段落到精致结论；连续对称纠偏；总结式高级感|
|段落密度|段落围绕服务动作、视线、证件/包/水杯等物件和父女动作切换|为了显得细腻而铺满环境；物件清单化|
|对白方式|S2 的对白要由追问、回避、端架子和动作打断推动；可以不顺，但必须有关系因果|完整问答讲清信息；人物准确说出真实动机|
|旁白方式|马雯雯只能短促误读、尴尬和社畜式自黑；旁白不承担真相解释|连续吐槽替代场景；替读者分析云水汇为什么怪|
|心理描写|压低心理分析，只保留短促误判、烦躁、尴尬和卡住|父女关系真相直说；“她意识到/真正的问题是”式总结|
|动作承载|父女关心、冷硬、不理解、陈科体面和云水汇过度服务优先落在动作与物件上|动作后立刻解释动机；把动作写成主题证明|
|环境描写|S1 只用服务动作、称呼、流程、家属安抚和物件错位表现高级得怪|空间异常、恐怖氛围、低级情色或脏感|
|幽默表达|笑点来自马雯雯错误框架、服务过度周到、陈科体面回避、父女动作冲突|段子手吐槽；固定 punchline；把人物写成讲笑话|
|去 AI 化重点|沿用导演层通用规则；本章额外加强“不要把高级得怪解释成设定，不要把父女关系解释成结论”|复制导演层规则全文；用备注式语言进入正文|

## 关键场景文字演出契约

|场景|文字质感|对白节奏|动作/环境用法|心理/旁白边界|禁止写法|
|---|---|---|---|---|---|
|S1 场外承接|轻快但不俏皮；服务好得让人难以归类|工作人员话少、勤快、体面；不解释规则|用水杯、外套、称呼、退半步、安抚家属等动作承载“高级得怪”|马雯雯可短促误读，不能连续吐槽或解说|低俗会所；恐怖空间；客服模板长句；显影诈骗|
|S2 父女冷硬|生活化、粗一点，关系压力要落地|陈艾艾短问；陈科转移、端架子、强调正规；问题被动作打断|用拿证件、挡人、扣包、站位、杯子等动作表现关心和抵抗|不直说“其实相爱”；马雯雯只看出尴尬和冷硬|温情泄露；伦理说教；陈艾艾像审讯机器；陈科像纯蠢老人|

## 对白自然化规则

|规则|适用场景|说明|
|---|---|---|
|现实问题被关系话术打断，再被动作拉回来|S2|陈艾艾问证件、陪同、能不能走；陈科可转成“态度、长辈、正规服务”，但动作要把问题拉回现实|
|工作人员少说话但服务频繁|S1|工作人员不讲完整服务模板；通过称呼、递东西、提醒、退开距离制造过度周到|
|马雯雯不承担解释功能|S1-S2|她可误读和短促嘴损，但不能替作者讲明云水汇为什么怪或父女为什么这样|
|父女对白不能清洁问答化|S2|问题要被顶回、躲开、动作打断或转成面子冲突|

## 去 AI 化禁止清单

只填写本章或关键场景新增、加强、例外化的禁止项。通用规则留在导演层可复用文本质量规则中。

|风险|具体禁止|替代方向|
|---|---|---|
|父女关系被解释成结论|禁止直接写“他们其实相爱”或等价总结|让陈艾艾不放手、拿证件、挡人、压火等动作承载|
|高级得怪被说明书化|禁止旁白解释“这里怪在服务过度精准”|用工作人员的动作、称呼、退让距离和陈科反应呈现|
|马雯雯吐槽过密|禁止连续用马雯雯段子压住父女压力|吐槽缩短，转回物件、动作或尴尬停顿|
|陈科广告员化|禁止让陈科完整夸云水汇或像代言人|让他说体面防御、被女儿追问后的回避话|

|检查项|通过标准|
