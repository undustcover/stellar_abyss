# C03《这事好像比会所还怪》

## 正文

云水汇门口太安静了。

马雯雯本来已经把脸皮准备好了。她想过工作人员会拦人，会躲，会装没看见，也想过马木一开口就把场面弄得像社区调解。

结果都没有。

陈科刚从侧门出来，旁边一个穿深色马甲的人就迎上去，手里端着杯热水，声音低得像怕惊着谁。

“陈先生，外面风凉，先拿着。”

陈科接过去，没立刻喝，先把杯子握住。

杯子不大，外面套了纸套，递过来的角度也正好。马雯雯看着那只手收回去，又看见对方往旁边让了半步，刚好不挡路，也不显得他们是被赶出来的。

这半步让她很不舒服。

不是凶，不是不正规。

是太会了。

陈科原本脸色还僵着，被这杯水一托，肩膀慢慢放平，嘴也有了地方硬起来。

“看见没有。”他说，“人家地方正规，家属来了也照顾。”

马雯雯差点去看马木。

她忍住了。

她怕自己一看，马木又摆出那种“你还小你不懂”的破表情。偏偏马木这会儿没说话，只盯着陈科手里的杯子，像那杯水里面藏着什么营业执照。

工作人员又把一件薄外套搭在椅背上。

“陈先生肩颈刚放松过，别直接吹风。几位也可以先坐一下。”

马雯雯脚尖往后缩了一点。

这服务范围有点宽。

会所还管肩颈吹不吹风吗。

陈艾艾没坐。

她站在陈科面前，连那件外套都没看一眼，手伸出去。

“证件。”

陈科眉头一皱：“什么证件？”

“身份证。”

“在我这儿。”

“给我。”

陈科把杯子换到另一只手，胳膊往里收了收：“你现在跟你爸说话就这个样？”

陈艾艾看着他。

她不吵，也不急着解释，手还停在那里，像陈科要是不拿，她就能一直停下去。

工作人员很轻地退远了一点。

马雯雯注意到了。

陈科也注意到了。

他脸上那点被女儿逼住的难看，立刻又被“外人在旁边”的东西撑起来。

“这么多人看着呢。”陈科说，“你别一来就弄得跟审犯人一样。”

“谁陪你进去的？”陈艾艾问。

陈科没答，先喝了一口水。

水应该不烫。他喝得很慢。

陈艾艾的手落下去，按在椅背上。

“谁。”

“工作人员。”陈科说，“人家有接待，有流程。你不要把什么都想得那么难听。”

“单独待过没有？”

“我一个大活人，坐一会儿还要跟你报备？”

“有没有。”

陈科把杯子放到旁边的小圆桌上，响声不大，脸色却沉了。

“陈艾艾，你现在是不是觉得你爸什么都不懂？”

这话出来，马雯雯后背紧了一下。

不是因为凶。

是这句话太像很多家里吵架前的那一下。菜还在锅里，钥匙还在门口，谁也没真正说重话，可那个方向已经偏了。

陈艾艾还是没软。

她伸手去碰陈科的外套内袋。

陈科一把按住：“干什么？”

“拿证件。”

“我自己不会拿？”

“那你拿。”

陈科没动。

两个人隔着那只内袋僵住。陈科按着自己的衣服，陈艾艾按着他的手腕，谁都没往回缩。

马雯雯看得有点难受，又有点尴尬。

她以前以为父女吵架，要么是女儿哭，要么是父亲骂。陈艾艾和陈科不是。一个冷着脸往里扣，一个端着架子往外顶，话都不多，力气全在手上。

马木这时候往前走了一小步。

陈艾艾没看他，只说：“爸，证件给我。”

这一声爸很硬。

硬得不像求。

陈科的手却松了一点。

他很快又把脸绷住，从内袋里摸出身份证夹，递到半路停了停。

“你拿了要干什么？”

“先放我这儿。”

“你这是不信我。”

陈艾艾接过身份证夹，没翻，直接扣进自己的包里。

拉链拉上的声音特别清楚。

“走。”她说。

陈科坐直：“去哪儿？”

“回家。”

“我这边还没完。”

“不用完了。”

“怎么不用？”陈科声音压低，反而更硬，“人家安排好了，我招呼都不打一声就走？像什么话？”

马雯雯这次没忍住，朝云水汇里面看了一眼。

里面灯光还是稳的，前台那边有人低头整理东西，没催，也没乱。像他们这边不是冲突，只是一个需要被妥善放好的家庭小问题。

这就更麻烦了。

如果它乱一点，脏一点，或者哪怕心虚一点，马雯雯都能找到词。

可它偏偏干净，体面，连递水都知道温度。

陈艾艾站到陈科和侧门中间。

“你现在跟我走。”她说。

陈科看着她，脸一点点涨红，不知道是气的，还是被女儿挡在外人面前下不来台。

马木终于开口：“先别进去。”

陈科立刻转头：“你也来管我？”

马木没接这句。

他看了一眼桌上的水，又看了一眼陈艾艾包里的证件位置，说：“先让她拿着。”

马雯雯心里咯噔一下。

她堂哥这语气太自然了。

自然得像这种地方的证件、杯子、家属、门口，他都见过处理流程。

陈科冷笑一声。

“行。”他说，“现在都懂，都比我懂。”

工作人员站在几步外，听见这句，也只是微微低了低眼，没有过来劝。

马雯雯忽然觉得，这地方最怪的不是它像不像会所。

是它把每个人都照顾得很有分寸。

照顾到陈科还能要脸。

照顾到家属不好翻脸。

照顾到她这个旁观的外人，连一句“不对劲”都说不出口。
